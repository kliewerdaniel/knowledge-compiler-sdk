# Sovereign Intelligence Stack

The Sovereign Intelligence Stack (SIS) is a local-first system for compounding
human judgment into deployable software. It treats knowledge as source code.

## 1. Principles

Intelligence is not the model. Intelligence is the accumulated decisions that
shaped the model. The stack prefers local-first everything, data sovereignty,
and reproducibility.

## 2. Layers

The stack is organised as five layers: SovereignSpec, Recipe Compiler, Signal
Router, Evaluation Loop, and Apprenticeship Engine. Each layer emits artifacts
that the next layer consumes.

## 3. Observability

If the loop is the product, observability becomes the operating system. The
Intelligence Observatory records tacit judgments so they can be reused.
