# pass-09-specifications — Specification Generation

**Produces:** `application-ir` (specification portion) · **Consumes:** `reasoning-ir`, `semantic-ir`, `graph-ir`
**Deterministic:** no · **Model required:** yes

Converts reasoned knowledge into a structured software spec: functional
requirements, the data model (entities/relationships from the graph), and the
capabilities a user needs. This is the bridge from *knowledge* to *software*.
Skills: `skills/specification-generation`, `skills/architecture-generation`.
